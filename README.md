# 🚀 AI Career Analyzer

An AI-powered employability prediction dashboard built using Machine Learning and Streamlit.

## 📌 Features
- Placement probability prediction
- AI-generated career insights
- Feature importance visualization
- Interactive dashboard UI
- Personalized recommendations

## 🛠️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Plotly

## 📊 Machine Learning Workflow
1. Data preprocessing
2. Exploratory Data Analysis
3. Feature engineering
4. Model training using Random Forest
5. Prediction & probability analysis
6. Interactive dashboard deployment

## 🎯 Project Goal
To analyze student employability readiness using academic, aptitude, project, and skill-based factors.

- Resume analysis
